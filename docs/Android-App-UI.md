# Android App UI

- Navigation Bar

- 我的

  - 右上角为一个邮件状的图标，为通知页面。

    - 公告

    - 过期提醒

  - 上部显示昵称、学号、ID。若尚未登录，则显示为未登录，点击进入登录页面。

  - 下部有几个栏目：
    - 常用联系方式
    - 我的商品（可以删、查、改）
      - 特别地，可以调整已售出数量。
    - 我的收购（可以删、查、改）
    - 设置（用 JSON 保存除账号以外的设置）
      - 账号设置（若尚未登录则跳转登录页面）
        - 修改昵称
        - 修改密码（经统一身份认证后可以修改）
        - 退出登录
        - 注销账号
      - LLM 设置（可以保存或清空）
        - 一个下拉列表，选择协议（先仅提供 OpenAI Compatible）
        - Base URL
        - API Key（提示不会上传服务器）
        - 模型名称
        - 超时阈值
        - 书籍识别提示词
        - 一般商品识别提示词
      - 语言设置（未来用于多语言）
      - 主题设置（未来用于更改颜色方案）
    - 关于（点击后跳转关于页）

- 买卖（分为“商品”和“收购”两个 tabs，分别显示商品列表和收购列表，两 tabs 的界面基本相同）
  - 顶部布置文本框和一个按钮，用于搜索。
  - 列表是动态加载的，现有项显示完后继续下滑加载更多。
  - 如果处于登录状态，每个项可以点进去查看详情。仅详情页显示交易地点和联系方式，列表不显示。
  - 右下角布置两个悬浮的按钮，从上到下分别是筛选和发布新项。
    - 点击筛选键唤起弹窗，设置筛选条件，包括标签。可以清空条件、取消或确认。
    - 点击发布新项，若登录则跳转进入发布页面，可以进行批量发布。若未登录则跳转登录页面。
      - 先选择是书籍还是一般商品，对商品类型分别展示不同的可填写字段。
      - 可以拍照或上传图片给 LLM 识别，每次上传一张图片，将结果合并到当前添加的商品/求购列表中，列表应暂存本地。一并提交。若未配置 LLM API 则跳转到配置页面。
      - 显示热门标签供选择，并允许用户自定义标签。标签个数上限在编译期配置。
